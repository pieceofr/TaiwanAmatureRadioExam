根據提供的來源文件，以下為文件中提到的電子元件與儀器名稱及其功能解釋：

### 一、 電子元件類

* **電阻器 (Resistor)**：在直流電路中用於阻止電流流動的元件 1, 2。  
* **電位計 (Potentiometer)**：一種可變電阻器，經常用作可調的音量控制，用於控制阻抗 1, 3。  
* **電容器 (Capacitor)**：由兩個或多個由絕緣體隔開的導電表面組成，能在電場中儲存能量 1, 2, 4。  
* **電感器 (Inductor)**：通常由線圈組成，能在磁場中儲存能量 1, 2, 4。  
* **二極體 (Diode)**：包含陽極 (Anode) 與陰極 (Cathode)，允許電流僅在一個方向流動的半導體元件 2, 3。  
* **發光二極體 (Light Emitting Diode, LED)**：常用作視覺指示器的元件，當電流流過時會發光 2, 3, 5。  
* **電晶體 (Transistor)**：由三層半導體材料製成，可用作電子開關或放大器，具有增益 (Gain) 能力 2, 3。常見類型包括：  
* **雙極性電晶體**：分為 **PNP** 或 **NPN** 型，電極為射極 (Emitter)、基極 (Base) 和集極 (Collector) 3。  
* **場效應電晶體 (Field Effect Transistor, FET)**：其三個電極為源極 (Source)、閘極 (Gate) 和汲極 (Drain) 2, 3。  
* **積體電路 (Integrated Circuit, IC)**：將許多半導體和其他設備結合到一個包裝中的設備 5。  
* **變壓器 (Transformer)**：用於將交流電 (Alternating Current, AC) 轉換為不同電壓（如將 120V 降至低電壓）的元件 3, 5。  
* **繼電器 (Relay)**：由電磁鐵控制的開關 5, 6。  
* **保險絲 (Fuse)**：用於保護電路免受電流過載影響的安全裝置 2, 7。  
* **斷路器 (Circuit Breaker)**：與火線串聯，在過載時自動中斷電源的安全設備 7。  
* **濾波器 (Filter)**：用於阻擋或通過特定頻率訊號的電路。包括低通 (Low-pass)、高通 (High-pass)、帶阻 (Band-stop) 與帶通 (Band-pass) 濾波器 8-10。  
* **鐵氧體扼流圈 (Ferrite choke)**：咬合在電纜上，用於防止射頻電流引起音頻失真或干擾 8, 10, 11。

### 二、 無線電設備類

* **收發機 (Transceiver)**：發射機 (Transmitter) 和接收機 (Receiver) 的組合單元 5, 12。  
* **中繼電臺 (Repeater)**：能自動轉發其他電臺訊號的設備，通常具有頻率偏移 (Offset) 設定 10, 13。  
* **射頻功率放大器 (Radio Frequency Power Amplifier)**：增加發射機低功率輸出的設備 14。  
* **前置放大器 (Preamplifier)**：安裝在天線和接收機之間，用於放大微弱訊號 10, 14。  
* **終端節點控制器 (Terminal Node Controller, TNC)**：連接在發射機與電腦之間，用於數據封包通訊 6, 15。  
* **混頻器 (Mixer)**：將無線電訊號從一個頻率轉換為另一個頻率的電路 5, 14。  
* **振盪器 (Oscillator)**：產生所需頻率訊號的電路 2, 14。  
* **變頻器 (Transverter)**：將特定頻段（如 28 MHz）的輸出轉換為另一頻段（如 222 MHz）的設備 14。  
* **天線 (Antenna)**：用於輻射或接收電磁波。文件中提到的類型包括偶極天線 (Dipole)、八木天線 (Yagi)、指向天線 (Directional) 及橡皮天線 (Rubber duck antenna) 3, 16, 17。

### 三、 測量儀器類

* **歐姆表 (Ohmmeter)**：測量電路中的電阻 6, 7, 18。  
* **電壓表 (Voltmeter)**：測量電位或電動勢 (Electromotive Force, EMF)，連接時需與電路並聯 6, 7。  
* **電流表 (Ammeter)**：測量電流強度，連接時需與電路串聯 4, 6。  
* **萬用表 (Multimeter)**：一種多功能儀器，通常用於測量電壓和電阻 18。  
* **駐波比計 (Standing Wave Ratio Meter, SWR Meter)**：串接於發射機與天線之間，用於衡量負載與傳輸線的匹配情形 14, 15。  
* **天線分析儀 (Antenna Analyzer)**：用於確定天線是否諧振在所需的工作頻率 14。  
* **方向功率表 (Directional Power Meter)**：除 SWR 表外，可用於確定饋線與天線是否正確匹配的儀器 6。  
* **假負載 (Dummy Load)**：由無感電阻器和散熱器組成，用於測試時防止信號輻射 6, 14。  
* **電表 (Meter)**：在數字刻度上顯示訊號強度 (S-meter) 的儀器 5。
