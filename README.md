# 台灣業餘無線電二級與三級考試學習包
## CQ CQ CQ
本工具旨在協助記憶業餘無線電人員執照考題（二級／三級）。使用簡單的網頁程式, 並透過將題庫數位化並引入權重記憶演算法，幫助考生更有效率地完成備考。

**不僅僅是無線電工具**：
程式靈感源自廣受好評的 Anki 學習法。如果你有其他想背的內容，只要修改 config.js 並匯入自訂的 CSV 題庫，就能瞬間秒變專屬於你的客製化記憶 App。靈活度極高，歡迎大家動手玩玩看！

## 禮包內容
* **記憶加強法考題測試 APP**：本地端 HTML 程式，無需聯網。 
    + (快速瀏覽: https://youtu.be/UZRtBrPuFm0)

* **備考閱讀筆記**：由 AI 整理之「題目-答案-解說」與補充圖卡。
  > **⚠️ 僅供參考，請以法規為準**
* **二級題庫**：新增六科二級分科題庫與 NCC 原始題庫 PDF。
* **總題庫 (570 題)**：完整考題彙整。
* **原始資料檔**：所有考題均提供 CSV 格式，方便二次利用。

> **安全性說明**：本程式為純本地端執行之 HTML 與 CSV 檔案，不具備聯外功能，原始碼公開透明。

## APP 使用說明
1.  **選擇題庫與題數**：使用者可自訂練習範圍與題目數量。
2.  **權重標記**：答題後根據難易度給予標籤（EASY / GOOD / HARD / HELL）。
    * **EASY**：已掌握，該題不再出現。
    * **其他標籤**：影響後續出現機率（權重演算法）。
3. **影片教學** :  https://youtu.be/UZRtBrPuFm0

## 參考學習法 (Divide＆Conquer)
1.  **了解原理**: 學習參考資源裡的 [BV3FG 初學者頁面]提供很好的整體解說.
2.  **預習答案解答**：優先閱讀 APP/Explain的「Ans_分類筆記與補充」，建立正確答案的第一印象。
3.  **少題做起**：優先處理「射頻干擾」、「電磁相容性」、「安全防護」等符合題目少的題目。
4.  **出題率高**：練習「無線電通訊方法」，其出題率相對較高。
5.  **基礎理論區**：複習「無線電系統原理」，運用基本電學基礎。
6.  **很多要強記的**：無線電規章與相關法規。
