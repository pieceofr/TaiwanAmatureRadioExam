# 三級無線電考試學習包
## CQ CQ CQ
本工具旨在協助記憶三級業餘無線電人員執照考題。透過將題庫數位化並引入權重記憶演算法，幫助考生更有效率地完成備考。

## 禮包內容
* **記憶加強法考題測試 APP**：本地端 HTML 程式，無需聯網。
* **總題庫 (570 題)**：完整考題彙整。
* **分科題庫**：將考題拆分為六大考組，落實「分而治之」（Divide and Conquer）策略。
* **備考閱讀筆記**：由 AI 整理之「題目-答案-解說」與補充圖卡（僅供參考，請以法規為準）。
* **原始資料檔**：所有考題均提供 CSV 格式，方便二次利用。

> **安全性說明**：本程式為純本地端執行之 HTML 與 CSV 檔案，不具備聯外功能，原始碼公開透明。
> **閱讀說明** : 備考閱讀筆記, 只要雙擊上面文件就可以在本網站閱讀歐! 如果下載到自己的電腦, 需要用支援 md 的軟體開啟, 比如瀏覽器.

## 測驗 APP 使用說明
1.  **選擇題庫與題數**：使用者可自訂練習範圍與題目數量。
2.  **權重標記**：答題後根據難易度給予標籤（EASY / GOOD / HARD / HELL）。
    * **EASY**：已掌握，該題不再出現。
    * **其他標籤**：影響後續出現機率（權重演算法）。
3. 如何下載  (如果沒在用 git)
   + 下載 zip 檔:
    點進右上方有一個綠色按鈕 (<>Code). 按下去, 選擇 Download ZIP 選項, 就會出現下載對話框.
    下載完成後解壓縮ZIP檔. APP 檔案夾裡有 RadioTest.html 即為 APP.

## 學習路徑建議 (Divide and Conquer)
1.  **預習筆記**：優先閱讀「Ans_分類筆記與補充」，建立正確答案的第一印象。
2.  **快速取分區**：優先處理「射頻干擾」、「電磁相容性」、「安全防護」等符合直覺的題目。
3.  **核心加強區**：練習「無線電通訊方法」，其出題率相對較高。
4.  **基礎理論區**：複習「無線電系統原理」，運用基本電學基礎。
5.  **法規攻堅區**：最後衝刺「無線電規章與相關法規」。
